# Airline Disruption Analysis Dashboard

## Overview
This project analyzes the impact of the Israel-US and Iran conflict on airline operations using an interactive Power BI dashboard.

The dashboard focuses on operational disruption indicators such as flight cancellations, route rerouting, delay trends, and affected air corridors to understand how geopolitical events influence the aviation sector.

## Tools & Technologies
- Power BI
- Data Cleaning & Transformation
- KPI Analysis
- Interactive Dashboarding

## Key Insights
- Identified patterns in flight disruptions and delay trends during the conflict period
- Analyzed rerouting behavior and affected aviation corridors
- Used KPI tracking, slicers, and drill-through reporting for deeper operational analysis

## Files Included
- Power BI dashboard file (.pbix)
- Dashboard screenshots
- Supporting dataset (if applicable)

## Purpose
This project demonstrates the use of business intelligence and data visualization to study a real-world geopolitical disruption and its operational impact on the aviation industry.
